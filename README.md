## Requirements

*  Node.js.
*  npm package manager (last version)
*  angular cli


## Install

On your terminal run commands below

```bash
# Clone application
git clone https://github.com/PedroHenriqueDevBR/wallet-app.git

# Enter the application folder
cd wallet-app/frontend

# Install dependecies
npm install

# Enter the backend folder
cd ../backend

# Install backend dependecies
npm install

# Run backend
npm run start

# Run application
ng serve --open 

```
